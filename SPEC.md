과업지시서
KNOWLEARN 인력 전략 AI 플랫폼 구축
1. 사업 개요
1.1 과업 명
온톨로지 기반 인력 전략 시뮬레이션 AI 플랫폼 구축

1.2 추진 배경
기업은 인력·프로젝트·원가 데이터를 보유하고 있으나,

데이터는 분산되어 있고

질문은 복합적이며

미래 전략 판단은 사람에 의존

본 과업은 정형 스키마 → 온톨로지 → 하이브리드 RAG → 전략 시뮬레이션으로 이어지는
차세대 인력 의사결정 플랫폼을 구축하는 것을 목표로 한다.

2. 과업 목표
정량 목표
기업별 상이한 DB 스키마를 설정 기반으로 흡수

다음 분기 인력 전략 시뮬레이션 3가지 이상 시나리오 제공

자연어 기반 인력 질문 응답 정확도 90% 이상

정성 목표
HR툴이 아닌 경영 의사결정 도구

“조회”가 아닌 선택의 결과 비교

3. 전체 시스템 범위
[기업 DB]
   ↓
[Schema Mapping / Ontology Builder]
   ↓
[Knowledge Graph + Vector Index]
   ↓
[Hybrid RAG Engine]
   ↓
[AI Agent + Strategy Simulator]
   ↓
[Dashboard / Admin / API]
